# Focus Stacking

Building project.

```bash
git clone git@github.com:gooeyone/focus-stacking.git
cd focus-stacking
mkdir build
cd build
cmake ../
make
```

To run tests:
```bash
./runAllTests
```

To run focus stacking program:
```bash
./focusStacking [images_directory]
```

Where `images_directory` argument is optional and by default is equal to `input`.
